## Checklist

- [ ] Update all `TODO:`
- [ ] Update all `package-name` occurences
- [ ] Add a LICENSE.md at the root
- [ ] Have gh cli installed
- [ ] Work on the integration logic
- [ ] Work on docs
  - [ ] JSDoc
  - [ ] Package README
- [ ] Format/lint (`pnpm lint:fix`)
- [ ] Setup GitHub repository
- [ ] Release (see below)
- [ ] Share!

To create a release:
- [ ] `pnpm run changeset`
- [ ] Commit and push
- [ ] `pnpm release`
